# 亖米Mesh网关集成

这是一个用于Home Assistant的亖米Mesh网关集成，支持通过串口和TCP连接网关，自动发现和控制亖米Mesh网络中的智能设备。

## 功能特性

- 多种连接方式 (串口115200波特率 / TCP网络连接)
- 设备自动发现和配网
- 设备状态持久化
- 实时状态同步
- 完整设备控制

## 支持的设备类型

- 零火开关(1-6路)、单火开关(1-4路)
- 智能插座
- 智能灯(调光、色温)
- 智能窗帘
- 温湿度传感器
- 门磁传感器、人体感应
- 温控器
- 情景面板、插卡取电

## 安装方法

1. 通过HACS安装或手动复制文件到custom_components目录
2. 重启Home Assistant
3. 在集成页面添加"亖米Mesh网关"
4. 选择连接方式并配置相应参数
5. 开启设备发现模式进行配网

## 配置要求

- Home Assistant 2023.1.0+
- Python 3.11+
- pyserial 3.5+ (串口连接时需要)
- 串口连接或TCP网络连接到亖米网关设备

## 技术规格

- 协议版本: 亖米Gateway Protocol V1.0
- 通信方式: 串口 (115200 bps, 8N1) 或 TCP网络连接
- 支持设备: 11种设备类型，50+子类型
- 网络拓扑: Mesh网络，最大支持105个设备
- 响应时间: < 200ms
- 数据持久化: 自动保存设备配置
